TeamFlow is a production-ready project management SaaS built with a **shared backend** consumed by **React Web** and **Flutter Mobile** clients.

## Features

- JWT Authentication (Access + Refresh tokens)
- Role-based Authorization
- Multi-tenant Projects & Tasks
- Real-time updates using WebSockets
- Activity logs (audit trail)
- Pagination & search
- Protected frontend routing

## Tech Stack

### Frontend (Web)

- React + TypeScript
- Vite
- TanStack Query
- Tailwind CSS

### Backend

- Node.js + Express
- TypeScript
- PostgreSQL
- Prisma ORM
- Socket.IO

### Mobile

- Flutter (in progress)
