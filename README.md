# TeamFlow

TeamFlow is a full-stack project management SaaS application built using React, Node.js, and PostgreSQL.

## Tech Stack

- Frontend: React, TypeScript, Vite
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Auth: JWT (Access + Refresh Tokens)

## Goal

Build an industry-ready application with clean architecture, authentication, authorization, and scalable APIs.
